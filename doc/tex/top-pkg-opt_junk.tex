
The standard library {\tt opt\_junk} package implements support for the 'opt' facility for dis/including optional Mythryl C-library bindings e.g. those for OpenGl and Gtk.

The {\tt opt\_junk} package implements the \ahrefloc{api:Opt\_Junk}{Opt\_Junk} api.

The {\tt opt\_junk} package source code is in \ahrefloc{src/lib/make-library-glue/opt-junk.pkg}{src/lib/make-library-glue/opt-junk.pkg}.
