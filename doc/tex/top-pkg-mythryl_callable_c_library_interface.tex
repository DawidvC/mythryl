
The standard library {\tt mythryl\_callable\_c\_library} package implements support for Mythryl access to C-level library functionality, e.g. Posix-library functionality.

The {\tt mythryl\_callable\_c\_library} package implements the \ahrefloc{api:Mythryl\_Callable\_C\_Library}{Mythryl\_Callable\_C\_Library} api.

The {\tt mythryl\_callable\_c\_library} package source code is in \ahrefloc{src/lib/std/src/unsafe/mythryl-callable-c-library-interface.pkg}{src/lib/std/src/unsafe/mythryl-callable-c-library-interface.pkg}.
