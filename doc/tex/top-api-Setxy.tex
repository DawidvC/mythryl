
The standard library {\tt Setx} api defines an interface for set implementations with two type parameters ('X', 'Y').

The {\tt Setxy} api is implemented by the 
\ahrefloc{pkg:red\_black\_setxy\_g}{red\_black\_setxy\_g} package.

The {\tt Setxy} api source code is in \ahrefloc{src/lib/src/setxy.api}{src/lib/src/setxy.api}.
