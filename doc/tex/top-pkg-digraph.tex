
The standard library {\tt digraph} package implements fully persistent directed graphs.

The {\tt digraph} package implements the \ahrefloc{api:Digraph}{Digraph} api.

The {\tt digraph} package source code is in \ahrefloc{src/lib/src/digraph.pkg}{src/lib/src/digraph.pkg}.

See also:  \ahrefloc{pkg:digraphx}{digraphx}.
