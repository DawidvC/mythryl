
The standard library {\tt code\_segment} package implements the code segment representation used by the Mythryl compiler.

The {\tt code\_segment} package implements the \ahrefloc{api:Code\_Segment}{Code\_Segment} api.

The {\tt code\_segment} package source code is in \ahrefloc{src/lib/compiler/execution/code-segments/code-segment.pkg}{src/lib/compiler/execution/code-segments/code-segment.pkg}.
