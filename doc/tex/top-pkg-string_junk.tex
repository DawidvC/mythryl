
The standard library {\tt string\_junk} package implements extended support for strings, such as {\tt atoi}, {\tt atod}, {\tt basename}, {\tt dirname}, {\tt trim}.
 
The {\tt string\_junk} package implements the \ahrefloc{api:String\_Junk}{String\_Junk} api.

The {\tt string\_junk} package source code is in \ahrefloc{src/lib/std/src/string-junk.pkg}{src/lib/std/src/string-junk.pkg}.

See also:  \ahrefloc{pkg:string}{string}.
