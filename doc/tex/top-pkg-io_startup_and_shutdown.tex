
The standard library {\tt io\_startup\_and\_shutdown} package implements support for automatically closing open I/O streams on exit.

The {\tt io\_startup\_and\_shutdown} package implements the \ahrefloc{api:Io\_Startup\_And\_Shutdown}{Io\_Startup\_And\_Shutdown} api.

The {\tt io\_startup\_and\_shutdown} package source code is in \ahrefloc{src/lib/std/src/io/io-startup-and-shutdown.pkg}{src/lib/std/src/io/io-startup-and-shutdown.pkg}.
