
The standard library {\tt digraphxy} package implements fully persistent directed graphs with two type variables ('X', 'Y') for node and edge contents.

The {\tt digraphxy} package implements the \ahrefloc{api:Digraphxy}{Digraphxy} api.

The {\tt digraphxy} package source code is in \ahrefloc{src/lib/src/digraphxy.pkg}{src/lib/src/digraphxy.pkg}.

See also:  \ahrefloc{pkg:digraph}{digraph}.
