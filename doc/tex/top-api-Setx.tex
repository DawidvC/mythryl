
The standard library {\tt Setx} api defines an interface for set implementations with one type parameter ('X').

The {\tt Setx} api is implemented by the 
\ahrefloc{pkg:red\_black\_setx\_g}{red\_black\_setx\_g} package.

The {\tt Setx} api source code is in \ahrefloc{src/lib/src/setx.api}{src/lib/src/setx.api}.
