
The standard library {\tt bounded\_queue} package implements queues with a maximum length, implemented by dropping entries when the max length is achieved.

The {\tt bounded\_queue} package implements the \ahrefloc{api:Bounded\_Queue}{Bounded\_Queue} api.

The {\tt bounded\_queue} package source code is in \ahrefloc{src/lib/src/bounded-queue.pkg}{src/lib/src/bounded-queue.pkg}.
