
The standard library {\tt issue\_unique\_id} package implements issuing unique Id values, useful for identifying records since record addresses can change 200 times per second due to the garbage collector.  In practice most code will instead use the version implemented in \ahrefloc{src/lib/core/init/pervasive.pkg}{src/lib/core/init/pervasive.pkg}.

The {\tt issue\_unique\_id} package implements the \ahrefloc{api:Issue\_Unique\_Id}{Issue\_Unique\_Id} api.

The {\tt issue\_unique\_id} package source code is in \ahrefloc{src/lib/src/issue-unique-id.pkg}{src/lib/src/issue-unique-id.pkg}.
