
The standard library {\tt planfile} package implements hashtables specialized to integer keys.
reads one or more  .plan files like {\tt src/opt/gtk/etc/gtk-construction.plan},
parses and validate the contents, and returns them as a list of paragraphs.

The {\tt planfile} package implements the \ahrefloc{api:Planefile}{Planfile} api.

The {\tt planfile} package source code is in \ahrefloc{src/lib/make-library-glue/planfile.pkg}{src/lib/make-library-glue/planfile.pkg}.

See also:  \ahrefloc{pkg:planfile\_junk}{planfile\_junk}.
