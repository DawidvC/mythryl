
The standard library {\tt runtime} package implements access to the Mythryl system runtime.

The {\tt runtime} package implements the \ahrefloc{api:Boxed\_Runtime}{Boxed\_Runtime} api.

The {\tt runtime} package source code is in \ahrefloc{src/lib/core/init/runtime.pkg}{src/lib/core/init/runtime.pkg}.
