
The standard library {\tt Typelocked\_Rw\_Matrix} api defines an interface mutable matrices specialized to a particular type.

The {\tt Typelocked\_Rw\_Matrix} api is implemented by the 
\ahrefloc{pkg:rw\_matrix\_of\_one\_byte\_unts}{rw\_matrix\_of\_one\_byte\_unts} package.
\ahrefloc{pkg:rw\_matrix\_of\_eight\_byte\_floats}{rw\_matrix\_of\_eight\_byte\_floats} package.

The {\tt Typelocked\_Rw\_Matrix} api source code is in \ahrefloc{src/lib/std/src/typelocked-rw-matrix.api}{src/lib/std/src/typelocked-rw-matrix.api}.
